Operadores Lógicos: La Lógica del "Y", "O" y "NO"

Los operadores lógicos nos permiten combinar varias condiciones en una sola. Son la clave para crear reglas de decisión más potentes y flexibles. Los tres operadores lógicos principales son AND, OR y NOT.

Operador AND ( `&&` )

El operador AND (y) devuelve `true` únicamente si todas las condiciones que une son verdaderas. Si al menos una es falsa, el resultado completo es falso.

En la siguiente imagen, se muestra un código que solo concede acceso VIP si la persona es mayor de 21 años Y además tiene un pase válido.

En este caso, ambas condiciones (`edad > 21` y `tienePase == true`) deben cumplirse para que se ejecute el bloque del `if`.

Operador OR ( `||` )

El operador OR (o) devuelve `true` si al menos una de las condiciones que une es verdadera. Solo será falso si todas las condiciones son falsas.

Ejemplo Visual

El siguiente código otorga un descuento si es martes O si la persona es estudiante. Solo se necesita que una de las dos condiciones sea cierta.

Si cualquiera de las dos condiciones (`esMartes == true` o `esEstudiante == true`) es verdadera, el usuario recibirá el descuento.

Operador NOT ( `!` )

El operador NOT (no) invierte el valor de una condición. Si algo es `true`, lo convierte en `false`, y viceversa.
