Ejemplo #2: Acceso a un Evento

En este ejemplo, combinamos operadores lógicos para decidir si una persona puede entrar a un evento especial.

Las reglas para entrar son:

  • La persona debe ser mayor o igual a 18 años Y tener un ticket válido.
  • O, si tiene un pase VIP, puede entrar sin importar la edad.

Ejemplo Visual

A continuación se muestra el código que implementa esta lógica de acceso.

Análisis: usamos los paréntesis `()` para agrupar la primera condición `(edad >= 18 && tieneTicket)`. Esto asegura que esa condición se evalúe primero como un conjunto. Luego, el resultado de ese conjunto se evalúa con el operador `||` junto a la condición `esVip`. Si cualquiera de las dos partes principales es verdadera, el acceso es permitido.
