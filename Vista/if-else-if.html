El Condicional `if-else if-else` (Escalera de decisiones)

¿Qué pasa si tenemos más de dos caminos posibles? Aquí es donde `else if` se vuelve muy útil. Esta estructura nos permite encadenar múltiples condiciones. El programa las revisa una por una, desde arriba hacia abajo. En cuanto encuentra una que sea verdadera, ejecuta su código y se salta el resto. Si ninguna es verdadera, ejecuta el bloque `else` final (si existe).

Sintaxis Básica


if (primeraCondicion) {
    // Se ejecuta si primeraCondicion es verdadera
} else if (segundaCondicion) {
    // Se ejecuta si primeraCondicion es falsa Y segundaCondicion es verdadera
} else {
    // Se ejecuta si TODAS las condiciones anteriores son falsas
}

Ejemplo Visual

A continuación, se mostraría una imagen de un código que clasifica un número como positivo, negativo o cero.

En el código del ejemplo visual, el programa primero revisa si `numero > 0`. Si eso es falso, entonces y solo entonces, revisa la siguiente condición: `numero < 0`. Si ambas condiciones son falsas, se ejecuta el bloque `else` final, concluyendo que el número debe ser cero.
