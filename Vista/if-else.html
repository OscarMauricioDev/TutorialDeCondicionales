El Condicional `if-else` (Si... Si no...)

La estructura `if-else` nos permite tomar una decisión con dos caminos. Si la condición del `if` es verdadera, se ejecuta un bloque de código. Si es falsa, se ejecuta el bloque de código que está dentro del `else`. De esta manera, siempre se ejecuta una de las dos opciones.

Sintaxis Básica


if (condicion) {
    // Código a ejecutar si la condición es verdadera
} else {
    // Código a ejecutar si la condición es falsa
}

Ejemplo Visual

Aquí verías una imagen con un ejemplo de código que determina si un número es par o impar.

En un ejemplo como este, si una variable `numero` tuviera el valor `7`, la condición `numero % 2 == 0` daría como resultado `false`. Por lo tanto, el programa ignoraría el bloque `if` y ejecutaría el código dentro del bloque `else`, indicando que el número es impar.
