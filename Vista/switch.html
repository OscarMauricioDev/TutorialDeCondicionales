El Condicional `switch` (Selector de Casos)

La estructura `switch` es una alternativa a la escalera `if-else if-else`. Es especialmente útil cuando tienes una sola variable que puede tener múltiples valores específicos y quieres ejecutar un código diferente para cada valor.

El `switch` evalúa una expresión, compara el resultado con los valores de cada `case` y ejecuta el bloque de código asociado al `case` que coincida. La palabra clave `break` es muy importante, ya que detiene la ejecución y evita que se sigan evaluando los demás casos. El bloque `default` es opcional y se ejecuta si ningún caso coincide.

Sintaxis Básica


switch (expresion) {
    case valor1:
        // Código a ejecutar si expresion === valor1
        break;
    case valor2:
        // Código a ejecutar si expresion === valor2
        break;
    default:
        // Código a ejecutar si no coincide ningún caso
}
