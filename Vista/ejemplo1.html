Ejemplo #1: Sistema de Calificaciones

En este ejemplo, adaptaremos la lógica para un sistema de calificaciones, donde la calificacion va de 0.0 a 5.0.

Las reglas son simples y se resuelven con un `if-else`:

  • Si la nota es mayor o igual a 3.0, el estudiante está APROBADO.
  • Si la nota es menor a 3.0, el estudiante está REPROBADO.

Ejemplo Visual

A continuación se muestra el código que implementa esta lógica de calificación.

Análisis del código: Primero, es una buena práctica validar que la nota se encuentre dentro del rango esperado (entre 0.0 y 5.0). Si la nota es válida, se aplica una única condición `if (nota >= 3.0)`. Si esta condición es verdadera, se aprueba; si es falsa, se ejecuta el bloque `else` y se reprueba.
