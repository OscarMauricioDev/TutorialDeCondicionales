El Condicional `if` (Si...)

La estructura `if` es la más simple de todas. Se utiliza para ejecutar un bloque de código solamente si una condición específica es verdadera. Si la condición es falsa, el código dentro del `if` se ignora por completo y el programa sigue su curso normal.

Sintaxis Básica


if (condicion) {
    // Código a ejecutar si la condición es verdadera
}

Ejemplo Visual

A continuación se muestra una imagen de cómo se vería una función en JavaScript que verifica si la edad de un usuario es mayor o igual a 18 y muestra un mensaje de bienvenida.

En este código, la línea `alert("¡Bienvenido! Eres mayor de edad.");` solo se ejecutará si el valor de la variable `edad` es 18 o más.
